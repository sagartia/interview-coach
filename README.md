# AI 面試教練

模擬技術面試的 AI 應用，支援即時評分與改善建議。

## 功能

- 輸入目標職位，自動生成對應技術問題
- 即時評分（0-100分）並分析優缺點
- 提供參考答案重點
- 連續多題面試流程

## 技術架構

| 層級 | 技術 |
|------|------|
| 後端 | Java 21、Spring Boot 3.5、Spring AI |
| 前端 | Vue 3、Vite |
| AI 模型 | GPT-4.1-nano（透過 GitHub Models） |


## 使用方式

1. 輸入目標職位（例如：Java工程師、前端工程師）
2. AI 出題後在下方輸入回答
3. 按 Ctrl + Enter 送出，取得評分與建議
4. 繼續回答下一題
